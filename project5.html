1. Semester exam project

The final project for the first semester. This project has two facets. One: A complete redesign of my portfolio website. Two: Updating all of the content in the portfolio.

This project is big and very time consuming, which is why we got ca. 3 weeks to do it. In that time I have planned, designed and made my new portfolio website. Following some of the old styles with simplicity and to an extent colours. I must say that I am very pleased with the result.

Reflection of learnings

What I have learned

During this project I have learned very much. Becoming more and more familiar with HTML, CSS, javascript and jQuery is very satisfying. I did also experience the importance in having a clear thought out plan, how to do things and especially when to do them.

What I expected to learn

I really only expected to become more familiar with all the different tools and methods for planning, designing and making a website. And my expectations were definetly met. I am very pleased to have gone through such an exciting learning process.

What more do I need to learn

As always, there is plenty more to learn. But if I have to mention anything, then it is to truly learn the scope of possibilites within programming through HTML and CSS in particular. It seems endless with possibillities, but is hard to grasp the vast opportunities in these codes.
