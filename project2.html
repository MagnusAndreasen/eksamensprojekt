Reflection of learnings

What I have learned

With this project I struggled to learn HTML and CSS languages and as a result of that, my end product was very dissapointing. Nevertheless, I got basic hands on knowledge about programming and improved considerably, considering I wasn't very good at the start of this project.

What I expected to learn

I expected to learn the coding languages quicker and to be better at implementing the design to the website. This wasn't the case and I had to adapt my expectations to what I actually was able to do with the website.

What more do I need to learn

During this project I realised that this coding stuff doesn't come easy and you have to work for it to be good at it. So the languages have to be relentlessly trained as often as possible. And that is what I have to do to be as good as I want to be.
