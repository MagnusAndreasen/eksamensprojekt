Build the Sweetbot website

The fourth project this semester was a group assignment and the project was to make a responsive website for the company Sweetbot Design.

Our group were very productive and came up with a simple and great design. The content was quickly planned and each of us 3 in the group, got different roles and assignments. This was done to be as effective and productive as possible. We are very pleased with our final product and we got some great feedback after our presentation.

Reflection of learnings

What I have learned

During this project we learned about project management, teamwork and almost all of the earlier themes, for example HTML, CSS, making Icons, making photos for the website and so on.

What I expected to learn

For this project I was expecting to learn about how to be in a team when working on a project. I did learn this, so I am pleased.

What more do I need to learn

I am interrested in learning more about project management, because, as a person I am always planning and structuring everything I do, so this process is very interesting for me.
